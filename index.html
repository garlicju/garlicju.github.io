수업 특징

  • 진단
    첫 상담에서 현재 수준·목표·학습 습관을 점검
  • 루틴
    개념 정리 → 숙제 → 오답 → 피드백 사이클 고정
  • 관리
    과제량/시간표/취약 단원 추적 (필요 시 주간 리포트)

커리큘럼 예시

4주 플랜

  1. 레벨 진단 + 핵심 개념 재정비
  2. 빈출 유형 훈련 + 오답 템플릿
  3. 서술형/응용 강화
  4. 실전 모의 + 약점 보완

수업 방식

  • 수업 전: 과제/질문 수집
  • 수업 중: 풀이 구조화 + 즉석 변형
  • 수업 후: 오답 정리/추가 과제 안내
